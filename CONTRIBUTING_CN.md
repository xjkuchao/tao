# 贡献指南 (Contributing to Tao)

感谢您有兴趣为 Tao 做贡献！

## 如何贡献

1.  在 GitHub 上 **Fork 本仓库**。
2.  将您的 Fork **克隆 (Clone)** 到本地。
3.  为您的更改 **创建一个分支 (Branch)**。
4.  **进行更改** 并提交。
    *   请遵循代码风格 (Rustfmt)。
    *   为新功能或错误修复添加测试。
5.  将您的分支 **推送到您的 Fork**。
6.  向原仓库的 `main` 分支 **提交 Pull Request (PR)**。

## 报告问题

如果您发现 Bug 或有功能请求，请在 GitHub 上提交 Issue。

## 开发流程

*   本项目大量使用 AI 辅助开发 (Gemini, Claude, Codex 等)。
*   所有 AI 生成的代码都需要人工监督和审查。
*   我们的目标是用纯 Rust 复刻 FFmpeg 的功能。
