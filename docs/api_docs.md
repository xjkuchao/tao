# API Documentation

The full API documentation is available on [docs.rs](https://docs.rs/tao) (once published).

## Core Concepts

*   **Tao Core**: Basic types and utilities.
*   **Tao Codec**: Encoders and decoders.
*   **Tao Format**: Container formats (muxers/demuxers).
*   **Tao Filter**: Audio/Video filters.
